# imposter_checking

一個現代化、美觀且實用的客服異常掛斷檢測工具 (Call Center Anomaly Analytics Dashboard)。

## 🚀 功能特點
- **純前端運作**：100% 在本地瀏覽器中解析與分析數據，不經過任何伺服器，資料隱私安全。
- **惡意掛斷檢測**：依據「進線通話且通話時間極短（預設 < 10 秒）且客服人員主動掛斷 (Agent Ended)」的黃金規則自動抓取異常案件。
- **動態控制台**：可即時調整通話秒數閾值與其他篩選條件，數據與圖表即時更新。
- **數據看板與排行榜**：透過數據卡片與 Chart.js 圖表直觀展示每位客服的異常率（異常件數/總進線數），迅速識別 outliers（如異常率偏高的客服）。
- **異常明細檢索與匯出**：支援關鍵字全域搜尋，並可一鍵匯出篩選後的異常案件明細為 CSV 檔案。

## 📦 安裝與使用方式
1. 將本專案下載或 Clone 到本地。
2. 雙擊打開 `index.html` 檔案即可在瀏覽器中使用。
3. 拖放您的話務明細報表（Excel 格式 `.xlsx`）至上傳區域以開始分析。
